日志里面不记录静态文件：

	在访问日志里，过滤掉一些图片、js、css类的请求日志。因为这样的请求日志没有多大用，而且会占用很大的磁盘空间

如何配置？
	
	在虚拟主机配置文件里增加配置：

        location ~* \.(png|jpeg|gif|js|css|bmp|flv)$
        {
	    access_log off;
         }
 

补充：

	tail -f /data/logs/bbs.access.log  //-f选型可以动态查看一个文件的内容
	> 可以清空一个文件内容
	~* 表示不区分大小写的匹配  后面跟正则表达式   .表示任意一个字符
